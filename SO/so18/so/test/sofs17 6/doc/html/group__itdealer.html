Detailed Description

inode table dealer: mediates access to the disk inode table and to the inodes

This module guarantees that only a single copy of every inode is in memory, thus improving consistency.

Remarks
In case an error occurs, every function throws a SOException
Author
Artur Pereira - 2016-2017
