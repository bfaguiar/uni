Detailed Description

The SOFS17 mounting tool.

It provides a simple method to integrate the SOFS15 file system into Linux.

SINOPSIS: 

                mount_sofs15 [OPTIONS] supp-file mount-point
             OPTIONS:
               -d       --- set debugging mode (default: no debugging)
               -l depth --- set log depth (default: 0,0)
               -L file  --- log file (default: stdout)
               -h       --- print this help.
